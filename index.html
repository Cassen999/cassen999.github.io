About Me

My name is Cassen Gerber. I am a 29 year old full stack software engineer from Minneapolis Minnesota. I got into development while in my previous career as a barber when I realized, after trying a few different scheduling apps, there were not really any that were specific towards barbering. Seeing this huge market gap, I decided to pursue software engineering to hopefully help out the shop that I left. When I am not working on websites or software, you will probably find me playing video games with friends, watching Friends with my wife, or trying to keep my two cats (Winry and Spirit) from destroying our house.

My Honeymoon Gallery

This is a small gallery of my honeymoon trip to Kyoto and Tokyo Japan. If you click on the images, the images are swapped with the picture's description and vise versa. The like button is functional as well and will add a like to the photo of your choosing. This project was done using JavaScript, React and HTML and utilizes both a client and server.

Server-side Calculator

This project I made is a calculator. It does the basic addition, subtraction, multiplication and division. This was done with Node.js, JavaScript, Bootstrap and HTML. All calculations are done in the server and sent to the client

Database To Do List

This application is a full stack movie database app. It utilizes a database to store movies along with a synopsis, genres, and a poster. It also allows a user to add a movie to the database, and when a movie is clicked, it will route the user to a page that displays the poster along with the description of the chosen movie. This project was done using SQL, JavaScript, Material-UI, Node.js, React, React-Redux, and Redux-Sagas.
